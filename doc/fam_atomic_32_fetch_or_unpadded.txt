fam_atomic_32_fetch_or_unpadded(3)
==================================
:doctype: manpage

NAME
----
fam_atomic_32_fetch_or_unpadded - unpadded 32 bit fam atomic fetch or operation

SYNOPSIS
--------
....
#include <fam_atomic.h>

int32_t
fam_atomic_32_fetch_or_unpadded(int32_t *address,
                                int32_t arg);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 32 bit fam
atomic variable, reads the contents of the atomic variable,
performs bitwise OR between the atomic variable and 'arg', and
returns the original value that was read.

RETURN VALUE
------------
Returns the previous value of the fam atomic (before the atomic
variable was OR'd with 'arg'.
