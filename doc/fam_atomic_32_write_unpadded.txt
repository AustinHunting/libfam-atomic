fam_atomic_32_write_unpadded(3)
===============================
:doctype: manpage

NAME
----
fam_atomic_32_write_unpadded - unpadded 32 bit fam atomic write

SYNOPSIS
--------
....
#include <fam_atomic.h>

void
fam_atomic_32_write_unpadded(int32_t *address,
			     int32_t value);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 32 bit fam
atomic variable, sets the atomic variable to 'value'. This is
essentially an atomic swap that does not return a value.

RETURN VALUE
------------
None
