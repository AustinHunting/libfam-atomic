fam_atomic_64_swap_unpadded(3)
==============================
:doctype: manpage

NAME
----
fam_atomic_64_swap_unpadded - unpadded 64 bit fam atomic swap

SYNOPSIS
--------
....
#include <fam_atomic.h>

int64_t
fam_atomic_64_swap_unpadded(int64_t *address,
			    int64_t value);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 64 bit fam
atomic  variable, reads the contents of the atomic variable,
sets the variable to 'value', and returns the original
value that was read.

RETURN VALUE
------------
Returns the previous value of the fam atomic (before it
was set to 'value').
