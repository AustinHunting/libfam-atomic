fam_spin_unlock_unpadded(3)
===========================
:doctype: manpage

NAME
----
fam_spin_unlock_unpadded - Releases an unpadded fam spinlock

SYNOPSIS
--------
....
#include <fam_atomic.h>

void
fam_spin_unlock_unpadded(struct fam_spinlock_unpadded *lock);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'lock', which is a pointer to an unpadded fam spinlock, releases
the lock that was previously acquired from fam_spin_lock_unpadded(lock)
or a successful fam_spin_trylock_unpadded(lock).

RETURN VALUE
------------
None
