fam_atomic_64_read_unpadded(3)
==============================
:doctype: manpage

NAME
----
fam_atomic_64_read_unpadded - unpadded 64 bit fam atomic read

SYNOPSIS
--------
....
#include <fam_atomic.h>

int64_t
fam_atomic_64_read_unpadded(int64_t *address);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 64 bit fam
atomic variable, reads the contents of the atomic variable and
returns that value. 

RETURN VALUE
------------
Returns the value of the fam atomic that was read.
